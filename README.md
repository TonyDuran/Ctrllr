# Ctrllr

"Controller" | A better `ctrl+r` for bash

## About

Ctrllr is a Rust-based CLI tool that improves upon the traditional bash `ctrl+r` command history search. It provides:

- **Version controlled history**: Store your command history in git
- **Taggable commands**: Tag and organize your most-used commands
- **Shareable across devices**: Sync your history across all your machines
- **Interactive TUI**: Built with ratatui for a better search experience
